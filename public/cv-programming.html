Projects delivered

The 2050 Calculator

An online tool that allows people to explore the future of the UK's energy system and has been in continuous operation since 2010. I wrote all the code, mostly in Ruby and Javascript with some C. Key features:

  • Code that takes an Excel spreadsheet and translates it into an equivalent model written in C.
  • Code to interface between the C library and the javascript in the browser
  • Bespoke javascript controls and charts.

New Media Knowledge

A bespoke content management written for the New Media Knowledge organisation in operation 2007-2014. 90% of the code was written by me, using Ruby on Rails. The final styling was done by a design firm. It involved:

  • User management, blog posts and event listings.
  • A mailing list system, including archives.
  • A ticketing system, including payment system integration.
  • It was written in Ruby on Rails, backed by MySql.

Talks.cam

An events listing site written for the University of Cambridge. It has been in continuous use since 2006. I wrote the front and back-end code, using an early version of Ruby on Rails. It involved:

  • User management, including interfacing to the early University of Cambridge Single Sign On system.
  • A clever but complex alternative to tagging (talks could appear on any number of lists. Lists could contain other lists.)
  • Providing content that could be embedded on other people's websites
